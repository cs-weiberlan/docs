## O que é a API Conta Simples

A **API Conta Simples** é a interface de integração da Conta Simples. Através dela, parceiros e sistemas externos se conectam à nossa plataforma de forma segura, padronizada e escalável.

Com a API Conta Simples você pode:

- **Consultar transações** de cartão de crédito e conciliar com seus sistemas
- **Gerenciar cartões** (físicos e virtuais) e seus dados
- **Baixar comprovantes** e documentos anexados às transações
- **Automatizar fluxos financeiros** do seu negócio

## Princípios da API

<AccordionGroup>
  <Accordion title="RESTful e previsível" icon="route">
    Seguimos convenções REST padrão. Recursos são substantivos, verbos HTTP indicam ações, e respostas seguem estruturas consistentes.
  </Accordion>
  <Accordion title="Segurança em primeiro lugar" icon="lock">
    Autenticação via tokens, comunicação exclusiva por HTTPS, e controle granular de permissões por escopo.
  </Accordion>
  <Accordion title="Idempotência" icon="repeat">
    Operações críticas suportam chaves de idempotência para garantir que retentativas não causem duplicações.
  </Accordion>
  <Accordion title="Versionamento" icon="code-branch">
    A API é versionada para garantir estabilidade. Mudanças breaking são comunicadas com antecedência e documentadas no changelog.
  </Accordion>
</AccordionGroup>
