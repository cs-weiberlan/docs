## Sobre esta seção

A **API Reference** é a referência técnica dos endpoints da API Conta Simples. Aqui você encontra:

- Todos os endpoints disponíveis com parâmetros e schemas
- Exemplos de request e response
- Códigos de resposta HTTP
- Playground interativo para testar chamadas

<Warning>
**Novo por aqui?** Recomendamos começar pelos [Guides](/comece-aqui/quickstart) antes de explorar a referência técnica. Os guias explicam os **conceitos e fluxos** — esta seção foca nos **detalhes técnicos** de cada endpoint.
</Warning>

## Autenticação

Todas as requisições requerem autenticação via **Bearer token** no header `Authorization`:

```bash
curl -X POST https://api-sandbox.contasimples.com/credit-card \
  -H "Authorization: Bearer {TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{...}'
```

| Header | Valor | Obrigatório |
|--------|-------|-------------|
| `Authorization` | `Bearer {TOKEN}` | ✓ |
| `Content-Type` | `application/json` | ✓ (para POST) |

## Convenções

### Datas

| Contexto | Formato | Exemplo |
|----------|---------|---------|
| Parâmetros de entrada | `YYYY-MM-DD` | `2025-01-15` |
| Campos de resposta | ISO 8601 | `2025-01-15T14:30:00.000Z` |

### Valores monetários

Valores vêm como `number` (decimais) em **Reais (BRL)**.

### Paginação

Paginação baseada em cursor via `nextPageStartKey`. Veja [Boas Práticas → Paginação](/guias/boas-praticas#paginação) para implementação completa.
